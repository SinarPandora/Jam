# 果酱 3

基于语法简洁的中文步骤定义语言 SXDL ，可由用户添加行为的 QQ 机器人

> QQBot  
> Based on Mirai Console + Mirai CQHTTP Plugin + PicqBotX
> 全面迁移到 Mirai 的工作正在进行……
